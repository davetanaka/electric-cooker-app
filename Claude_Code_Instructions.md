# Claude Code への追加実装指示書（Step 4: 品質チェックとデプロイ準備）

## 現在の状況
なんと全ての主要機能（比較・フィルター、製品詳細、AIチャット）の実装が完了しました！開発の進行スピードと品質、素晴らしいです。
ここからは、最後のブラッシュアップとVercelへのデプロイに向けた準備（Phase 4）に入ります。

## 今回のフェーズでの指示（Step 4: 最終確認とデプロイ準備）

以下の作業を行ってください。

### 1. UXとレスポンシブデザインの最終確認
- 各ページ（`/`, `/compare`, `/products/[id]`, `/chat`）が、モバイル（スマートフォン）からPCサイズまで崩れず綺麗に表示されるか確認し、必要に応じてTailwindのクラス（`sm:`, `md:` など）を調整してください。
- ユーザーにとって直感的なナビゲーション（戻るボタンやヘッダーリンク等）が備わっているか確認してください。

### 2. OGPタグとメタ情報の設定
- アプリケーショントップや製品ごとのページに対して、SEOとSNSシェア用に `<meta>` タグ（Title, Description, OGP）を適切に設定してください。
- PRDにある「カタログに載っていない情報も含めた、消費者のための横断比較」といったキーメッセージを盛り込んでください。

### 3. Vercel デプロイ準備
- `package.json` のビルドスクリプト（`npm run build`）がエラーなく通ることをローカルで確認してください（TypeScriptの型エラーやESLintの警告解消を含む）。
- AIチャットで使用している環境変数 `ANTHROPIC_API_KEY` は、Vercelのダッシュボード側で設定されるべきものであることを README.md に追記してください。

### 4. 完了報告
- ビルドが正常に通り、コード品質が万全であることを確認したら最終報告を行ってください。
その後、ユーザー（人間）側でVercelとGitHubを接続し、デプロイを行います。
