# Claude Code への初期実装指示書

## 目的
あなたは、電気調理鍋の横断比較Webアプリの開発を行うメインエンジニア（実装担当）です。
このリポジトリにある `PRD_Electric_Cooker_Comparison_App.md` を読み込み、要件に従ってプロジェクトのセットアップと初期実装を開始してください。

## 前提条件・テックスタック
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **AI API**: Claude API (将来的にチャット機能で使用)
- **Hosting**: Vercel (デプロイ前提)

## 開発の進め方ルール
1. **ステップバイステップの実装**: 一気に作ろうとせず、区切りの良い機能ごとに実装・動作確認を行ってください。
2. **Git Commit**: 機能単位でこまめにコミットしてください。
3. **日本語での記述**: コード内のコメントやドキュメントはすべて日本語で記述してください。
4. **プレースホルダー禁止**: 完全なコードを記述し、「残りは実装してください」といった省略は避けてください。
5. **PRDへの準拠**: 常にPRDの要件（特に情報ポリシーや中立性）を意識して実装してください。

## 今回のフェーズでの指示（Step 1: プロジェクト基盤の構築）

まず以下の作業を行ってください。ステップごとに完了したらユーザーに報告し、次の指示を仰ぐか自主的に進めてください。

### 1. Next.jsプロジェクトの初期化
- カレントディレクトリにNext.js (App Router, TypeScript, Tailwind CSS) のプロジェクトをセットアップしてください。
- （コマンド例: `npx create-next-app@latest . --typescript --tailwind --eslint --app` など）

### 2. shadcn/uiの導入
- プロジェクトにshadcn/uiを初期設定してください (`npx shadcn-ui@latest init` 等)。
- 共通で使いそうな基本コンポーネント（Button, Card, Input 等）があればインストールしておいてください。

### 3. トップページのベース作成
- PRDの「8.1 トップページ」および「8.2 比較・フィルター画面」を参照し、ベースとなるページのルーティングと静的なトップページ（ヒーローセクション、ナビゲーション）を作成してください。
- 魅力的なUIを意識し、モダンで洗練されたデザインの基盤を作ってください。

### 4. 完了報告
- 基盤が整い、開発サーバー (`npm run dev`) でトップページが正常に表示される状態になったら、作業内容をまとめ報告してください。
